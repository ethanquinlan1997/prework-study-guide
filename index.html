HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Stands for "Hypertext Markup Language"
  • A standardized system for tagging text files to achieve font, color, and graphics
  • All HTML pages have a series of HTML elements, consisting of a set of tags and attributes
  • HTML elements are the building blocks of a webpage

CSS

  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet
  • CSS stands for Cascading Stlye Sheets
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files
  • CSS is designed to enable the separation of presentation and content, including layout, colors, and font
  • A padding indicates how much space we want around the content inside an element
  • A margin indicates how much space we want around the outside of an element

    Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git branch [branch name]: creates a new branch at the current commit
  • git log: show all commits in the current branches history
  • git clone [url]: retrieve an entire repository from a hosted location via URL
  • git push [alias]/[branch]: Transmit local branch commits to the remote repository branch
  • git pull: fetch and merge any commits from the tracking remote branch

    JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • Array is a single variable that is used to hold a related group of data
